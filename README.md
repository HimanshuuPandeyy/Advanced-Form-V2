# Advanced HTML Form

A modern, interactive HTML form built with semantic HTML, input validation, and responsive design.

## Features
- Full Name, Email, Age, Gender, and Bio inputs
- Dropdown selection
- Textarea for user bio
- Validation for required fields
- Responsive layout
